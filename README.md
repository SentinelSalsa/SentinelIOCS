# SentinelIOCS
IOCS ive put into a sentinel friendly csv and JSON format that can be fed directly into Microsoft sentinels threat intelligence platform. 

(CSV ONLY) For Hashes select 'file indicators' under filetypes for IP's and Domains select 'all other indicators'

Future uploads will be in JSON format to integrate into sentinel threat intelligence go to threat intelligence> import > import using file> file format JSON > upload the JSON file you've got from here.

JSON files are generated using iocforge.com converter.

Sources: ESET, CISA, Deep instinct, SOC radar, SOC Radar, various reports, Internal.
